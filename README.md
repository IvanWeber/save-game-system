# Save System Demo (HTML5 + JavaScript)

A simple browser-based save/load system for games.

## Features

- 10 save slots  
- Timestamp and X position shown per slot  
- Save, load, delete with confirmation  
- Export/import all saves as `.json`  
- Clean menu UI
